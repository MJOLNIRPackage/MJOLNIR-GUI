MJOLNIR_Gui is a developed GUI application dedicated to the MJOLNIR package. It is meant to provide an interface to many of the features found in the software suite.

Some icons by Yusuke Kamiyamane. Licensed under a Creative Commons Attribution 3.0 License. See <a href='https://p.yusukekamiyamane.com/'>yusukekamiyamane.com</a>

Resolution calculations based on the original code from TAKIN 
<a href='https://doi.org/10.5281/zenodo.4117437'>https://doi.org/10.5281/zenodo.4117437</a>
